Frequently Asked Questions

Everything you need to know about using Pixaroid.

Privacy & Security

No. All processing runs entirely in your browser using the Canvas API and Web Workers. Your files never leave your device. Pixaroid has no backend server that receives files.
Never. Images exist only in your browser's memory while you process them. When you close the tab, they are gone. No cookies, no database, no cloud storage.
Yes. Because no personal data is ever collected or transmitted, Pixaroid is inherently GDPR-compliant. We have no data to delete, no cookies to consent to, no accounts to manage.

Usage & Formats

JPEG, PNG, WebP, HEIC/HEIF, GIF, BMP, TIFF, AVIF, SVG, and ICO are all supported as input. Output can be JPEG, PNG, WebP, GIF, PDF, or plain text (for OCR).
Up to 20MB for images and up to 50MB for PDF files. Larger files can cause performance issues depending on your device. For bulk tools, up to 100 images per batch are supported.
Yes — Pixaroid works in all modern mobile browsers including Chrome for Android and Safari on iOS. You can tap to browse files, or long-press to paste from clipboard. No app download required.
Yes — Pixaroid is a Progressive Web App (PWA). After your first visit, you can install it on your home screen and use all tools without an internet connection. The Service Worker caches all pages and workers.

PDF Tools

Browser-based PDF editing cannot re-flow existing text (this requires server-side rendering). You can cover existing text with a Whiteout block and add new text on top — the same approach used by Sejda and ilovepdf in browser mode.
Up to 50 pages for PDF to image conversion and OCR. For compress, merge, split, rotate and watermark, all pages are processed regardless of count.

Pricing & Limits

Yes — completely free. No account, no credit card, no subscription, no watermarks, no file limits. All 95+ tools are free forever.
Yes. The core tool set will always be free. If we ever introduce premium features, the free tier will remain fully functional with no watermarks or limits.
